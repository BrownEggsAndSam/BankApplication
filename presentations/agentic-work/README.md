# Agentic Work Presentation Package

This folder contains the portable HTML presentation package for the agentic-work and business-enablement discussion.

## Contents

- `Agentic-Work-Presentation-Package.zip` — all three HTML presentations, shared presentation assets, the primary 18-slide narrative, speaker notes, screenshot manifest, source blueprint, voice-memo transcript, and work-computer handoff instructions.

## Using the package

1. Download and extract the ZIP file.
2. Open `index.html` in a browser.
3. Start with `from-buzzwords-to-better-work.html`.
4. Read `handoff/WORK-COMPUTER-HANDOFF.md` before adding internal terminology or approved work screenshots.

The package intentionally contains no credentials, database connections, personal images, or production-system access.
